Про нас

Я — веб-розробник-самоук, який зараз навчається в Університеті інформаційних технологій. Створюю сучасні та функціональні веб-рішення: від лендінг пейджів до багатосторінкових сайтів. Мої послуги включають розробку базових і бюджетних дизайнів, підключення форм до Telegram, посадку сайтів на домен та базовий JavaScript для інтерактивності. Працюю з HTML, CSS, SCSS та JavaScript, створюючи сайти, які допомагають бізнесу зростати.

Моя мета — надавати якісні та доступні рішення для клієнтів по всьому світу. Якщо ви хочете стильний і ефективний сайт для свого бізнесу, заповніть форму або натисніть кнопку нижче!

Наші послуги

Веб-сайти

Лендінг пейджі для швидкого запуску вашого бізнесу онлайн.
Прості сайти для малого бізнесу чи особистих проєктів.
Багатосторінкові сайти для компаній із розширеним функціоналом.
Адаптивна верстка для коректного відображення на всіх пристроях.

Дизайн

Базові дизайни для швидкого старту за доступною ціною.
Бюджетні кастомні дизайни для унікального вигляду сайту.
Робота з HTML, CSS, SCSS для створення сучасного інтерфейсу.

Інтеграція та підтримка

Підключення форм до Telegram для зручного збору заявок.
Посадка сайту на домен для повноцінного запуску.
Базовий JavaScript для додавання інтерактивних елементів.
Робота з клієнтами по всьому світу для глобального охоплення.

Портфоліо

Проект 1

Простий адаптивний HTML/CSS проєкт, створений під час навчання, із жартівливим вмістом. Демонструє базові навички верстки та роботи з текстом.

Проект 2

Створив адаптивний лендінг пейдж на тему "Minimalist Room" для демонстрації навичок верстки та дизайну. Сайт містить секції з описом проєкту, процесу дизайну та досягнень у сфері інтер’єрного дизайну.

Проект 3

Створив адаптивний лендінг пейдж для демонстрації навичок веб-розробки. Використані HTML, CSS і SCSS для сучасного дизайну та базовий JavaScript для інтерактивності (наприклад, анімація при скролі та бургер меню).
