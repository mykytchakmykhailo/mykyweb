Комп'ютерні мережі
Тема: Встановлення MikroTik RouterOS на VirtualBox
Мета роботи
Встановити MikroTik RouterOS на віртуальну машину VirtualBox.
Завдання
1. Дати характеристику програмного забезпечення: MikroTik RouterOS, VirtualBox, WinBox.
2. Дати визначення поняттям: PPTP, PPPoE, Broadcast, Proxy ARP, PAP, CHAP, Firewall, GRE, L2TP, IPsec, MS-CHAP, VPN, OpenVPN.
3. Визначити призначення портів 1723 і 443.
4. Описати процес встановлення MikroTik RouterOS на VirtualBox.
5. Продемонструвати процес запуску на WinBox.
6. Оформити звіт.
1. Характеристика програмного забезпечення
MikroTik RouterOS — спеціалізована мережева операційна система, розроблена компанією MikroTik. Базується на ядрі Linux і призначена для керування маршрутизаторами та іншими мережевими пристроями.
VirtualBox — безкоштовне програмне забезпечення для віртуалізації від компанії Oracle. Дозволяє створювати віртуальні машини та запускати на них різні операційні системи.
WinBox — офіційна графічна утиліта MikroTik для налаштування та адміністрування пристроїв під керуванням RouterOS.
2. Визначення основних понять
PPTP — Point-to-Point Tunneling Protocol — тунельний протокол для створення VPN-з'єднань,тунельний протокол точка-точка, що дає можливість компютеру встановлювати захищене зєднання з сервером за рахунок створення спеціального тунелю в стандартній, незахищеній мережі.
PPPoE — Point-to-Point Protocol over Ethernet — протокол авторизації через Ethernet.
Broadcast — широкомовна передача даних усім пристроям у мережі.
Proxy ARP — технологія відповіді на ARP-запити за інші IP-адреси, за допомогою цієї технології проксі-сервер у певній мережі відповідає на запити протоколу ARP для IP-адреси, яка не находиться в цій мережі.
PAP — Password Authentication Protocol — простий метод автентифікації.
CHAP — Challenge Handshake Authentication Protocol — безпечний метод автентифікації,широко розповсюджений алгоритм перевірки автентичності, що передбачає передачу не самого пароля користувача, а непрямих відомостей про нього.
Firewall — система захисту мережі, яка фільтрує трафік за правилами.
GRE — Generic Routing Encapsulation — протокол тунелювання розроблено компанією CISCO.
L2TP — Layer 2 Tunneling Protocol — тунельний протокол 2-го рівня.
IPsec — набір протоколів для шифрування та автентифікації IP-пакетів.
MS-CHAP — Microsoft версія протоколу CHAP.
VPN — Virtual Private Network — віртуальна приватна мережа.
OpenVPN — відкрита реалізація VPN з високим рівнем безпеки.
3. Призначення портів 1723 і 443
Порт 1723 (TCP) — використовується протоколом PPTP для керуючого каналу VPN-з'єднання.
Порт 443 (TCP) — стандартний порт HTTPS. Часто використовується VPN-протоколами (OpenVPN, SSTP), щоб трафік був схожий на звичайний веб-трафік.
4. Процес встановлення MikroTik RouterOS на VirtualBox
1. Створити нову віртуальну машину у VirtualBox.
2. Підключити образ MikroTik RouterOS (.iso) до віртуального приводу.
3. Запустити машину та вибрати всі пакети (натиснути “a”).
4. Після завантаження пакетів натиснути “i” для встановлення.
5. Після встановлення вийняти диск та перезавантажити машину.
6. Увійти під логіном admin та встановити пароль.
5. Підключення через WinBox
1. Запустити програму WinBox.
2. Перейти у вкладку Neighbors і знайти віртуальну машину.
3. Вибрати пристрій та натиснути Connect.
4. Увійти під логіном admin та паролем.
